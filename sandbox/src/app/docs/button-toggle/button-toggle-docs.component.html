UI Kit использует стандартный компонент Button toggle из бибилиотеки Angular Material, предлагая собственный набор CSS модификаторов для его стилизации.

При стилизации подразумевается что эелементы кнопок <mat-button-toggle> всегда объедены в группу через <mat-button-toggle-group> и соответственно модификаторы могут применятся только ко всей группе <mat-button-toggle-group>, а не к отдельно взятой кнопке.

Тематизация

Поскольку в стандартном компоненте поддержка свойства color пока не реализованна, для тематизации компонента предлагается использовать CSS модификаторы - __primary, __accent и __warn, играющие аналогичну роль.

Использование разделителей

В качестве разделителей можно использовать <mat-divider>. Цвет разделетиля наследуется из выбранной в <mat-button-toggle-group> цветовой темы. Если <mat-button-toggle-group> имеет горизонтальную ориентацию (задается по умолчанию), необходимо использовать вертикальный разделитель <mat-divider [vertical]="true">.

Размер

Компонент может иметь один из трех предопределенных размеров - маленький, средний (соответствует базовому размеру) и большой, задаваемых соответствующими модификаторами: __size-s, __size-m, __size-l.

Ширина

По умолчанию, компонент не имеет заданной ширины, она определяется автоматически исходя из размера содержимого. Добовление CSS модификатора __w-100 позволит растянуть его на всю ширину родительского элемента.
