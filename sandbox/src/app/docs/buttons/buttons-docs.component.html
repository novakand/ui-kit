UI Kit использует стадартные компоненты кнопок Angular Material, представляя возможности по их стилизации и модификации с помощью собственного набора CSS модификаторов. Вся подробная информация с описанием базовых компнонентов и доступным API представленна в официальной документации.

Типы кнопок

В данном разделе приводятся примеры использования конопок типа: mat-button; mat-raised-button; mat-flat-button; mat-stroked-button. Примеры кнопок с иконками mat-icon-button можно найти в разделе "Style / Icons"

Тематизация

Окраска кнопок в соответствии с использемой цветовой темой производится сдандартым путем, с помощью атрибута color. Возможны три стадартных варианта цветовой темой: primary, accent и warn.

Изменение размера

Кнопки могут иметь одит из трех предопределенных рамеров: маленький, средний (соответствует базовому размеру) и большой. Для изменения размера кнопки используются используются соответствующие CSS модификаторы, которые необходимо указать в атрибуте class компонента кнопки: __size-s, __size-m, __size-l.
