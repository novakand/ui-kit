Two month calendar позволяет пользователям выбирать дату или интервал дат, путем выбора даты из календаря.

Отображение календаря состоит из двух частей, стандартного поля ввода постоянно отображаемого на странице и всплывающего окна календаря, появляющегося при нажатии на него.

Календарь имеет два режима работы, задаваемых при помощи свойства rangeSelectMode:
false - режим выбора одной даты;

true - режим выбора диапазона дат, выбирается дата начала и дата конца диапазона (значение по умолчанию).

Входные и выходные параметры

Получить выбранные в календаре значения дат можно подписавшись на события selectedFromDateChange() и selectedToDateChange() возвращающих даты начала и конца диапазона соответственно. Two month calendar поддерживает также различные входные параметры с полным списокм которых можно ознакомиться на вкладке "PROPERTIES & METHODS".

Properties

Name Description
@Input()

@Input()

@Input()

@Input()

@Input()

@Input()
